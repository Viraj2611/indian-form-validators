# Changelog

## 1.0.0

- Initial release
- PAN card validator with taxpayer type detection
- Aadhaar validator with Verhoeff checksum, format, and mask utilities
- Indian mobile number validator with prefix normalization (+91, 91, 0)
- GST number validator with state code lookup and PAN extraction
- IFSC code validator with bank name and branch code extraction
- Indian pincode validator with postal zone detection
- UPI / VPA validator with handle recognition
- `IndianValidators` facade for single-import convenience
- 100% zero external dependencies (pure Dart)
